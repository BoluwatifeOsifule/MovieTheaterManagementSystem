# Movie Theater Management System (Windows Forms)

A simple *Windows Forms application* built with *C#* and the *.NET Framework* to manage movies, showtimes, and customer bookings in a theater setting.

---

## Features
- Add movies with *Title, **Genre, and **Duration*
- Display movies in a list
- Add showtimes for selected movies
- Book tickets with *Customer Name* and *Seat Selection*
- Save all data for persistence

---

## Technologies
- C#
- .NET Framework (Windows Forms)
- Visual Studio
